# Sunu Tawfekh — Boutique

Site e-commerce **Nuxt 3** : catalogue (API Orion), fiche produit, panier, favoris, checkout et pages légales.

## Prérequis

- Node.js 20+
- npm

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

## Production

```bash
npm run build
npm run preview
```

## Configuration

Copier `.env.example` vers `.env` et ajuster si besoin (source de données Orion, slug boutique, etc.). Les variables serveur `NUXT_ORION_*` ne sont pas exposées au client.

Voir aussi `docs/deployment.md` et `docs/api-contract.md`.
